# Installation

Enter the following command in terminal

    pip install git+https://github.com/varghesereji/pyTANSPEC.git@timeseries_extraction

Then the pipeline with it's supporting packages will be installed in the environment.


# General issues with the piepline installation

- Encountered error while trying to install the package readline.

Solution: Install gcc and gfortran compilers first, then try installing pyTANSPEC.
